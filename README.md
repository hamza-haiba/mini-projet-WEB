# 🏋️ GymFit Marrakech - Projet Programmation web 

Ce projet est un mini-site web pour une salle de sport

## 🚀 Fonctionnalités
- **Accueil** : Présentation dynamique avec Hero Section.
- **Programmes** : Affichage des tarifs avec CSS Grid.
- **Calculateur IMC** : Logique JavaScript pour le calcul de l'indice de masse corporelle.
- **Contact** : Formulaire avec validation des données en JS.
- **Responsive Design** : Adapté aux mobiles et tablettes.

## 🛠️ Technologies Utilisées
- HTML5 / CSS3 (Grid & Flexbox)
- JavaScript (Vanilla)
- Font Awesome (Icons)
- Google Fonts (Roboto)
